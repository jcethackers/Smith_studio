`FOR FRONTEND`
1. `cd frontend` 
2. Install dependencies: `npm install`,`npm i`,
3. Run the app:`npm run dev`
4. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key

`FOR BACKEND`
1. `cd backend`
2. Install dependencies: `npm install express cors`,`npm install nodemailer`,`npm install mongodb`
3. run th app: `node index.js`
4. Set the `MONGO_URI` in [.env](.env) to your MONGO_URI
